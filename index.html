Política de Privacidade - PromoKratz

O PromoKratz é um sistema que publica promoções automaticamente no Instagram.

Dados coletados

O aplicativo não coleta dados pessoais de usuários. Ele apenas utiliza a API do Instagram para publicar conteúdo em contas autorizadas.

Uso das informações

As informações são utilizadas exclusivamente para publicar promoções na conta do Instagram vinculada.
